# hex2rgb(a) && hex2hsl(a)
Is a simple web app that **converts color values** from *HEX* to *RGB(a)* and *HSL(a)* or *RGB(a)* to *HEX* and *HSL(a)*. It also **copies converted value to clipboard** and **previews it**.

URL: https://tgomilar.github.io/hex2rgb/
